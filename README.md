# To-do-App

A desktop to-do app UI designed around sticky notes to visually organize tasks and track progress.

## Overview
The app allows users to:
- Create tasks as sticky notes
- Track active tasks in a progress view
- Review completed tasks
- Reorder tasks with drag and drop

No login is required – the focus is on clarity and ease of use.

## Screens
- To-Do Wall
- Progress Page
- Completed Tasks
- Help Page

## Design Details
- Visual task management using sticky notes
- Clear task flow: To-Do → Progress → Completed
